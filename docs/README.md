## Project at a glance

- **Name**: Plain Dharma
- **Domain**: plaindharma.com (selected, not yet purchased)
- **Tagline**: *The Buddha's foundational teachings in plain modern English.*
- **License**: CC0 / public domain (dharma-gift tradition)
- **Source content**: `combined-suttas.md` (single file, six teachings)
- **Mission**: widest possible distribution of these teachings

## The six teachings

1. The Buddha's First Talk — *Dhammacakkappavattana* (Setting the Wheel in Motion)
2. The Buddha's Second Talk — *Anattalakkhana* (On Not-Self)
3. The Buddha's Third Talk: The Fire Sermon — *Ādittapariyāya*
4. On Loving-Kindness — *Mettā Sutta*
5. The Foundations of Mindfulness — *Satipaṭṭhāna Sutta*
6. How to Decide What to Believe — *Kālāma Sutta*
